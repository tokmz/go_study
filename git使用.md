# git 基本使用

### 检查Git版本

```sh
git --version 
```

### 设置Git config

```sh
git config --global user.name "aikzy"
git config --global user.email "aikzy@qq.com"

git config --global --list
```

### 初始化Git仓库

```sh
git init
```

### 仓库状态

```sh
git status
```

### 添加文件

```sh
git add example.txt    
```

用于将文件的改动添加到Git的索引区
